# Hash-Analyzer 
<img src="https://raw.githubusercontent.com/Mister-RdanM19/Hash-Analyzer/refs/heads/main/hash.png">
Hash-Analyzer adalah utilitas desktop yang dirancang untuk melakukan validasi integritas data dan analisis kriptografi. Fokus utama proyek ini adalah menyediakan alat yang ringan, efisien terhadap penggunaan sumber daya sistem (Hemat CPU), dan memiliki antarmuka minimalis untuk keperluan audit keamanan profesional.

## Penafian Hukum dan Etika
Perangkat lunak ini dikembangkan hanya untuk tujuan edukasi dan pengujian keamanan yang sah. Pengguna bertanggung jawab penuh atas segala tindakan yang dilakukan dengan menggunakan alat ini. Penggunaan terhadap target tanpa izin tertulis merupakan tindakan ilegal dan melanggar kebijakan etika keamanan siber.

## Spesifikasi Teknis
* Dukungan Algoritma: Kompatibel dengan standar MD5, SHA-1, SHA-256, dan SHA-512.
* Optimasi Efisiensi: Mengimplementasikan teknik pembatasan proses (throttling) setiap 200 iterasi untuk mencegah beban berlebih pada prosesor dan menjaga stabilitas suhu perangkat.
* Antarmuka Minimalis: Menggunakan tema gelap dengan antarmuka berbasis teks untuk memastikan fokus penuh pada log aktivitas.
* Multithreading: Memisahkan proses kalkulasi dari antarmuka pengguna agar aplikasi tetap responsif selama proses analisis berlangsung.
